## 🧠 Prompt Framework: SILO-X

All prompts follow the **SILO-X Framework** — professional SEO content architecture:

```
S — Seed Keyword   → ONE primary keyword per post, with search intent defined
I — Intent Mapping → Informational / Transactional / Local — determines structure
L — Long-form      → H1–H3 hierarchy engineered for Google crawlability
O — Optimisation   → Keyword placement rules, LSI terms, meta data requirements
X — eXecution      → Word count, FAQ format, CTA placement, internal linking rules
```

## 💡 Key Learnings

1. **Content clusters beat single posts** — 5 interconnected posts build topical authority that 1 post never could
2. **Intent mapping is the #1 SEO skill** — transactional vs informational vs local determines every structural decision
3. **FAQ sections = fastest path to featured snippets** — Google pulls answer boxes from FAQ-structured content
4. **Local SEO is won by natural repetition** — 3-5 mentions of "Vijayawada" per post signals local relevance
5. **Meta data is part of the content** — requiring it in the prompt ensures every post is Google-ready from draft one
6. **E-E-A-T must be engineered** — patient stories, doctor credentials, clinic history, transparent pricing must all be deliberately included
